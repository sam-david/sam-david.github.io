Ruby Method Blog: .rand

There are always times where you are going to some way to generate random numbers. We can find a variety of uses for these from a random roll of a die, to a word scrambler that might randomize the letters. Whatever it is, programmers know that the use of random numbers does come up here and there. Luckily, Ruby has a pretty easy method already in place that can produce pseudo random integers, floats, or binary strings. They say pseudorandom because no computer can produce truly random numbers by computation. This also means that to the casual observer the numbers will appear random but over time there is a pattern to the numbers based on the original seed, and you could eventually guess what the next number in the order is.

The easiest way to execute the rand function is shown below. This would generate a random number between 0 and 1.

rand() #=> 0.34242484593133105

If you would like a random integer from 0 to 10 you could type to following.

rand(10) #=> 5

How about a specific range, say from 5 through 9?

rand(5..9)#=> 7

So as we said these numbers aren't truly random, they all start at a seed. You can specify the seed by doing the following.

prng1 = Random.new(1234)
prng1.seed #=> 1234
prng1.rand(100) #=> 47
prng1.rand(100) #=> 83

From then on when you call a random number using that same seed value, you will get the same results for your random numbers. This can be useful, however if you want to call on the results of your random numbers later on in a program. See below the results are the same with the same seed.

prng2 = Random.new(1234)
prng2.seed #=> 1234
prng2.rand(100) #=> 47
prng2.rand(100) #=> 83
