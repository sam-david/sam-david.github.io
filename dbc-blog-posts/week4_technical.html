Ruby Method: .map

The enumerable method I decided to write a blog on is the .map method. This method is used to iterate over each element in an array or range and returns an array with the specified function you provided. For example, if you wanted to multiply every number in the array against itself you can write a .map method to do just that.

[1,2,3].map {|x| x * x} #=> [1, 4, 6]

You can also use .map on a range to perform similar tasks on a specific range of numbers

(1..5).map {|x| (x * x) + 2} #=> [3, 6, 11, 18, 27]

.map is also useful on an array of strings. Lets say you wanted to capitalize every element in an array

["sam","fred","david"].map {|x| x.capitalize} #=> ["Sam", "Fred", "David"]

A super clever thing that people like to do is pass in a callable proc object, with a method, to run on each element in the array. By doing this you can write your function on a separate line in the proc, then call it on your array using .map

triple = Proc.new{|n| n*3}
[1,2,3].map(&triple) #=> [3, 6, 9]

Lastly, if you want to call a method instead of a proc, you can do this by using the METHOD method (confusing but look below)

def triple(n)
n * 2
end
[1,2,3].map(&method(:triple)) #=> [3, 6, 9]

You can use .map destructively by placing a '!' at the end of it. This will modify the original array though so be careful before using. Here I also show you how you could convert numbers into strings using .map

number_string = [1,2,3,4]
number_string.map!(&:to_s) #=> ["1", "2", "3", "4"]
