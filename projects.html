Projects

Code Crush

Final project of DevBootcamp, a coding typing game with a full back-end using Node.js to save high scores. The game itself is built with Phaser.io. We chose Phaser.io because it is one of the most popular JavaScript based game engines for 2D games. We also use OAuth Login for Google and Facebook to allow users to login with these accounts. I was the team lead of this project which we made it about a week.

Tech Used:

  • M.E.A.N Stack
  • MongoDB
  • Express
  • AngularJS
  • Node.js
  • Phaser.io

Tesla Car Comparison

Using Edmunds.com's API (vast automotive database), this site compares the Tesla Model S to any car on the market across a variety of statistics. Inspired from my time as an employee at Tesla, I made this tool so people could see the costs and energy usage of the electric car versus their gas guzzlers. These were very common questions and concerns we would get from customers.

  • jQuery

In Development

Who To Follow

A major solo project during the DevBootcamp curriculum in which we were required to use OAuth and the Sinatra framework. I chose to make an app that gave you suggestions of people to follow on Twitter based on category. The app works by querying tweets with words related to that category and adding them to the database whenever a user clicks that category. From there it queries the DB for all the users related to that category sorted by popularity. This allows to DB to continue to grow each time a user clicks.

Tech Used:

  • HTML/CSS
  • JavaScript
  • Sinatra
  • OAuth
